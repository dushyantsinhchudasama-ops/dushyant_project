# Food Delivery Console Application

A console-based Java application demonstrating Core Java concepts, object-oriented design, SOLID principles, design patterns, file handling, exception handling, and JUnit testing.

## Structure
- `src/main/java/com/fooddelivery` - application source code
- `src/test/java/com/fooddelivery/test` - unit tests
- `data` - plain text persistence files

## Build
- Java 17
- Maven (optional)

Compile and run with Java:
```
javac -d out src/main/java/com/fooddelivery/**/*.java
java -cp out com.fooddelivery.Application
```

Run tests with Maven if available:
```
mvn test
```
